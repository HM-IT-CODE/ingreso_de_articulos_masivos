# Ingreso de Artículos Masivos

Este proyecto es una aplicación web desarrollada con React que permite la carga e ingreso masivo de artículos. Es ideal para empresas o usuarios que necesitan gestionar grandes volúmenes de productos de manera eficiente y sencilla.

## Características principales

- Interfaz intuitiva y moderna.
- Carga masiva de artículos mediante archivos o formularios.
- Visualización y gestión de los artículos cargados.
- Componentes reutilizables y estructura modular.

## Estructura del proyecto

```
├── public/
│   ├── index.html
│   └── ...
├── src/
│   ├── assets/           # Imágenes y recursos
│   ├── components/       # Componentes reutilizables (Dropzone, Header, Home)
│   ├── routes/           # Definición de rutas (Router.jsx)
│   ├── App.js            # Componente principal
│   └── index.js          # Punto de entrada de la app
├── package.json          # Dependencias y scripts
└── README.md             # Documentación del proyecto
```

## Instalación y ejecución

1. Clona el repositorio o descarga el código fuente.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia la aplicación en modo desarrollo:
   ```bash
   npm start
   ```
4. Abre tu navegador en `http://localhost:3000` para ver la app en funcionamiento.

## Tecnologías utilizadas

- React
- JavaScript (ES6+)
- HTML5 y CSS3

## Estructura de carpetas destacada

- `components/Dropzone.jsx`: Componente para carga de archivos.
- `components/Header.jsx`: Encabezado de la aplicación.
- `components/Home.jsx`: Página principal.
- `routes/Router.jsx`: Manejo de rutas internas.
